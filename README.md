# Calculator with keyboard support

Calculator for basic math operations.  
On mouse click or with keyboard.  

[Link](https://azzryel.github.io/Calculator)
 
###### Keyboard:

Operation | Key
------------ | -------------
add | a
subtract | s
multiply | x
divide | d
ac (all clear) | space
ce (clear entry) | backspace
dot | .
equal | enter
numbers | 0-9
