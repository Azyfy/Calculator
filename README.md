# Calculator with keyboard support

Calculator for basic math operations.
On mouse click or with keyboard.
 
Keyboard:
operation   -   key
+           -   a           (add)
-           -   s           (subtract)   
*           -   x           (multiply)
/           -   d           (divide)

ac          -   space       (all clear)
ce          -   backspace   (clear entry)
.           -   .           (decimal)
=           -   enter       (calculate)

numbers     -   0-9
